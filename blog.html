Difference between map(), filter(), forEach() and find() method.

The map, filter, forEach, and find methods are each used to loop over the array, but there are some differences in their functionality.

The map() method in JavaScript returns a new array, and the length of the returned array is always equal to the length of the original array. The original array is not changed.

In JavaScript, the filter() method returns a new array, and the length of the returned array may be smaller than the length of the original array. The original array is not modified.

The forEach() function returns no value. It is often used instead of a loop to iterate over an array. The original array is not changed.

On the other hand, the find method works like the filter() method but the find method returns the first data that matches the condition so if you want data based on a condition you can use the find method on the other hand if it is based on a condition. Filter method can be used if all the conditions associated with the condition are wanted

Difference between Local Storage and Session Storage

Local Storage

The data is stored in the user's browser until the local storage is deleted or cleared. Its storage capacity is 5-10 MB. Session Storage objects can be accessed using the sessionStorage read-only property. The difference between sessionStorage and localStorage is that localStorage data does not expire, whereas sessionStorage data is cleared when the page session ends.

What is Event Loop and How does it's work?

An event loop is a mechanism of JavaScript that always monitors the call stack and callback queue. The event loop continuously checks two data structures: the call stack and the message queue.

When the call stack is empty, the event loop checks the message queue for any tasks that are ready to be executed. The event loop adds the next task to the call stack and executes it, then continues to check the call stack and message queue in this way, until there are no more tasks to execute.

What is the purpose of quizTimer() & chosseQuiz() function

The quizTimer() function creates and updates a timer on a wep page for a quiz. It uses setInterval() function to execute a callback function that updates the timer every second.

The chooseQuiz() function records the user's answer. It adds a new answer to an array, depending on whether an answer already exists for the question. Finally, it calls a function to display all the recorded answers on the web page.
